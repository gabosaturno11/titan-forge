## Features

### Linguistic Console
- 4 Master Faders (F1, F3, F7, F10)
- Implicit Framing, Emotional Undercurrent, Confidence Emergence, Silence Placement
- Real-time console state export

### Voice System
- 8 Modes: Raw, Teacher, Prophet, Philosopher, Mystic, Rebel, Companion, Confessor
- 4 Preset Stacks: Gentle Authority, Trust-First, Non-Coercive, High-Precision
- Mode-specific fader configurations

### TRITON Onboarding
- 10-question identity sequence
- CEO lens (Sam Altman / First-Principles)
- Export to Notion

### APOAPSIS Writer
- Double-panel Markdown editor
- Live preview (Crimson Pro serif)
- Chapter management
- Export: MD, Full Manuscript

## Design
- Industrial dark theme (#050505)
- JetBrains Mono + Sora fonts
- No emojis
- Border-left accent system
- Glaze morphism effects

## Files
- `index.html` - Main Linguistic Console
- `onboarding.html` - TRITON sequence
- `writer.html` - APOAPSIS double-panel editor
- `logs.html` - Mission control

## Tech Stack
- Pure HTML/CSS/JS
- TailwindCSS (CDN)
- Marked.js (Markdown parser)
- LocalStorage state persistence
- Offline-first
