# SATURNO BONUS VAULT SETUP

## Files
- `bonus-vault-interactive.html` - The main interactive tool
- `Early_rise.wav` - Custom audio file for timer completion

## Setup Instructions
1. Place both files in the same directory
2. Open `bonus-vault-interactive.html` in a browser
3. Click the small dot (top-right) to enter edit mode
4. Password: `saturno2026`
5. Edit any text inline - skill names, exercises, etc.

## Audio Setup
The tool will try to load `Early_rise.wav` from the same directory.
If not found, it falls back to Web Audio API beeps.
